## 🎯 Как использовать

### 1. Открыть веб-интерфейс

Откройте в браузере: **http://localhost**

### 2. Загрузить документы

- Перетащите файлы в область загрузки (drag-and-drop)
- Или нажмите для выбора файлов
- Поддерживаемые форматы: PDF, JPG, PNG, TIFF
- Максимальный размер: 50MB

### 3. Выбрать форматы экспорта

Отметьте нужные форматы:
- ☑ TXT - простой текст
- ☑ DOCX - Word документ с таблицами
- ☑ XLSX - Excel с объединенными ячейками
- ☑ CSV - CSV файл (UTF-8 BOM)

### 4. Запустить обработку

Нажмите кнопку "Обработать" и дождитесь завершения

### 5. Скачать результаты

Нажмите на кнопки скачивания для каждого формата

## 📁 Что реализовано

### Backend API

✅ **Endpoints:**
- `POST /api/v1/upload` - загрузка файла
- `POST /api/v1/upload/batch` - пакетная загрузка
- `POST /api/v1/process` - запуск обработки
- `GET /api/v1/status/{task_id}` - статус задачи
- `GET /api/v1/result/{task_id}` - скачивание результата
- `GET /api/v1/health` - проверка здоровья

✅ **OCR движок:**
- Распознавание русского текста (PaddleOCR)
- Распознавание таблиц с объединенными ячейками
- Предобработка изображений (deskew, enhance)
- Извлечение и структурирование текста

✅ **Экспорт:**
- TXT - простой текст
- DOCX - Word с таблицами
- XLSX - Excel с объединенными ячейками
- CSV - UTF-8 с BOM

✅ **Асинхронная обработка:**
- Celery + Redis для очереди
- Отслеживание прогресса
- Пакетная обработка

### Frontend

✅ **Компоненты:**
- FileUpload - drag-and-drop загрузка

✅ **Функции:**
