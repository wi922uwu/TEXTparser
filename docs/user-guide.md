# Руководство пользователя OCR Web Service

## Введение

OCR Web Service — это инструмент для распознавания текста и таблиц из сканированных документов. Сервис работает полностью автономно, без подключения к внешним сервисам.

## Поддерживаемые форматы

### Входные форматы
- **PDF** — многостраничные PDF документы
- **JPG/JPEG** — изображения в формате JPEG
- **PNG** — изображения в формате PNG
- **TIFF/TIF** — изображения в формате TIFF

### Выходные форматы
- **TXT** — простой текст
- **DOCX** — документ Microsoft Word с таблицами
- **XLSX** — таблица Microsoft Excel
- **CSV** — таблица в формате CSV (UTF-8)

## Ограничения

- **Максимальный размер файла:** 50 МБ
- **Время обработки:** до 30 секунд на страницу
- **Поддерживаемые языки:** русский (приоритет), латиница
- **Рукописный текст:** не распознается (только печатный текст)

## Использование веб-интерфейса

### 1. Открытие приложения

Откройте браузер и перейдите по адресу:
```
http://localhost
```

Или по IP-адресу сервера:
```
http://192.168.1.100
```

### 2. Выбор форматов экспорта

В верхней части страницы выберите форматы, в которые нужно экспортировать результаты:

- ☑ **TXT** — для простого текста
- ☑ **DOCX** — для редактирования в Word
- ☑ **XLSX** — для работы с таблицами в Excel
- ☑ **CSV** — для импорта в другие системы

Можно выбрать несколько форматов одновременно.

### 3. Загрузка файлов

#### Способ 1: Перетаскивание (Drag & Drop)

1. Откройте папку с файлами
2. Выберите один или несколько файлов
3. Перетащите их в область загрузки на веб-странице
4. Отпустите кнопку мыши

#### Способ 2: Выбор через диалог

1. Нажмите на область загрузки
2. В открывшемся диалоге выберите файлы
3. Нажмите "Открыть"

### 4. Отслеживание обработки

После загрузки файлы автоматически отправляются на обработку.

В разделе **"Обработка"** отображается:
- Имя файла
- Текущий статус (PENDING, PROGRESS, SUCCESS)
- Прогресс-бар с процентом выполнения
- Текущий этап обработки:
  - Загрузка файла
  - Конвертация PDF
  - Предобработка
  - Распознавание текста
  - Экспорт результатов

### 5. Скачивание результатов

Когда обработка завершена, файл появляется в разделе **"Результаты"**.

Для каждого файла доступны кнопки скачивания в выбранных форматах:
- **TXT** — скачать текстовый файл
- **DOCX** — скачать документ Word
- **XLSX** — скачать таблицу Excel
- **CSV** — скачать CSV файл

Нажмите на нужную кнопку, чтобы скачать результат.

## Пакетная обработка

Вы можете загрузить несколько файлов одновременно:

1. Выберите все нужные файлы (Ctrl+Click или Shift+Click)
2. Перетащите их в область загрузки
3. Все файлы будут обработаны параллельно

**Рекомендация:** Не загружайте более 10 файлов одновременно для оптимальной производительности.

## Качество распознавания

### Факторы, влияющие на качество

**Хорошее качество:**
- Четкий печатный текст
- Высокое разрешение сканирования (300 DPI+)
- Прямое расположение текста (без наклона)
- Хороший контраст между текстом и фоном

**Плохое качество:**
- Размытые или нечеткие изображения
- Низкое разрешение (<150 DPI)
- Сильный наклон текста
- Рукописный текст
- Сложные фоны или водяные знаки

### Рекомендации по сканированию

Для лучших результатов:
1. Используйте разрешение 300 DPI или выше
2. Сканируйте в цвете или оттенках серого
3. Убедитесь, что документ расположен ровно
4. Избегайте теней и бликов
5. Используйте чистые, неповрежденные оригиналы

## Работа с таблицами

### Распознавание таблиц

Сервис автоматически определяет таблицы в документах и сохраняет их структуру:
- Строки и столбцы
- Объединенные ячейки
- Заголовки таблиц

### Экспорт таблиц

- **XLSX** — лучший формат для таблиц, сохраняет структуру и объединенные ячейки
- **DOCX** — таблицы вставляются в документ Word
- **CSV** — простой формат, объединенные ячейки дублируются
- **TXT** — таблицы представлены в виде текста с табуляцией

## Безопасность и конфиденциальность

### Хранение данных

- Загруженные файлы хранятся на сервере **24 часа**
- После этого автоматически удаляются
- Результаты обработки также удаляются через 24 часа

### Конфиденциальность

- Все данные обрабатываются **локально** на сервере
- **Нет передачи данных** во внешние сервисы
- **Нет подключения к интернету** во время обработки
- Полная изоляция в корпоративной сети

## Часто задаваемые вопросы

### Почему обработка занимает много времени?

- Многостраничные документы обрабатываются дольше
- Сложные таблицы требуют больше времени
- Высокое разрешение изображений увеличивает время обработки

### Что делать, если текст распознан неправильно?

1. Проверьте качество исходного документа
2. Попробуйте отсканировать с более высоким разрешением
3. Убедитесь, что текст печатный (не рукописный)
4. Проверьте, что документ не наклонен

### Можно ли обработать рукописный текст?

Нет, текущая версия поддерживает только печатный текст. Рукописный текст будет проигнорирован или распознан некорректно.

### Как обработать документ на другом языке?

Сервис оптимизирован для русского языка, но также поддерживает латиницу. Другие языки могут распознаваться с меньшей точностью.

### Что делать при ошибке обработки?

1. Проверьте формат файла (PDF, JPG, PNG, TIFF)
2. Убедитесь, что размер файла не превышает 50 МБ
3. Попробуйте загрузить файл снова
4. Обратитесь к администратору, если проблема повторяется

## Контакты поддержки

При возникновении проблем обратитесь к системному администратору.
